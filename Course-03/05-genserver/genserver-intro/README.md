# GenServer — Intro

Minimal introduction to `GenServer` in Elixir.

Defines a module that implements only the `init/1` callback. Shows how to start a GenServer process with `GenServer.start/2` and observe the initial state being set.

## Files

- `genserver_intro.exs` — Basic `GenServer` demo with a single `init/1` callback. Run with `elixir genserver_intro.exs`.
